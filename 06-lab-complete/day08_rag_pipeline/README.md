## Project Summary
- Topic: Vietnamese drug law and media coverage of entertainment figures related to drugs.
- Architecture: RAG pipeline with hybrid retrieval, BM25 lexical fallback, reranking, and generative answer synthesis.
- Highlights:
  - Data collection from legal documents and news articles
  - Document conversion to Markdown
  - Chunking and embedding indexing
  - Semantic search + lexical search
  - Reranking and citation-aware generation
